Matricular alumno

Para poder matricular a un alumno en un módulo profesional es necesario que pertenezcan al mismo curso. Seleccionamos el alumno y el módulo profesional a través de los componentes de la pantalla.

Una vez introducido todos los campos, solo deberemos pulsar el boton confirmar para realizar la operación.

El botón definido como volver, es empleado para volver a la pantalla anterior.

El botón definido como actualizar, se utiliza para poder realizar una búsqueda nueva si se ha cometido un error.

El botón definido como limpar campos, se utiliza para facilitar al usuario el borrado del contenido de las cajas de texto.

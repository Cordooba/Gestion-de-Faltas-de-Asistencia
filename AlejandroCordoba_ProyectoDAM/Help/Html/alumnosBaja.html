Baja de alumnos

Para poder dar de baja a un alumno en concreto, deberemos seleccionar el identificador del alumno para realizar la operación.

Una vez seleccionado el alumno, justamente abajo se nos mostrara la informacion del alumno a eliminar.

Para realizar la operación solo deberemos pulsar el botón de confirmar situado en la esquina inferior izquierda.

El botón definido como volver, es empleado para volver a la pantalla anterior.

Consultar faltas de asistencia

Para poder consultar las falta de asistencia de un módulo profesional, es necesario seleccionar un módulo profesional.

Si se selecciona algún valor se mostrará los alumnos y su número de faltas de asistencia correspondiente.

El botón definido como confirmar, es empleado para consultar el registro de una falta de asistencia, si ha superado el límite o no.

El botón definido como volver, es empleado para volver a la pantalla anterior.

Modificar faltas de asistencia

Para poder modificar una falta de asistencia, es necesario seleccionar un módulo profesional y un alumno que pertenezcan al mismo curso y este matriculado.

Si se selecciona algún valor definido se mostrará automáticamente debajo para mostrar la información correspondiente.

Selecionados ya los valores, por último deberemos modificar el número de horas.

El botón definido como confirmar, es empleado para modificar el registro de una falta de asistencia.

El botón definido como volver, es empleado para volver a la pantalla anterior.

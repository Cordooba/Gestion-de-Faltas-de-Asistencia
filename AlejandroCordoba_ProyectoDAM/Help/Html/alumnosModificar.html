Modificación de alumnos

Para poder modificar los datos de un alumno en concreto, deberemos introducir el identificador del alumno. Deberemos pulsar sobre el botón buscar para poder buscar un alumno con el identificador introducido en la base de datos.

Una vez que hemos encontrado al usuario automaticamente podremos visualizar todos sus datos para poder modificarlos. Solo deberemos pulsar sobre el botón confirmar, para poder guardar los valores modificados.

El botón definido como volver, es empleado para volver a la pantalla anterior.

El botón definido como limpar campos, se utiliza para facilitar al usuario el borrado del contenido de las cajas de texto.

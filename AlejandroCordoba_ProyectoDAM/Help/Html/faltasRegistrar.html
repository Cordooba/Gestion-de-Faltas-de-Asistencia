Registrar faltas de asistencia

Para poder dar de alta o registrar una nueva falta de asistencia, es necesario seleccionar un módulo profesional y un alumno que pertenezcan al mismo curso y este matriculado.

Si se selecciona algún valor definido se mostrará automáticamente debajo para mostrar la información correspondiente.

Selecionados ya los valores, por último deberemos definir el número de horas.

El botón definido como confirmar, es empleado para registrar una falta de asistencia.

El botón definido como volver, es empleado para volver a la pantalla anterior.

El botón definido como limpar campos, se utiliza para facilitar al usuario el borrado del contenido de las cajas de texto.

Baja de módulos profesional

Para poder dar de baja a un módulo profesional en concreto, deberemos seleccionar el identificador del módulo profesional para realizar la operación.

Una vez seleccionado el módulo profesional, justamente abajo se nos mostrara la informacion del módulo para eliminar.

Para realizar la operación solo deberemos pulsar el botón de confirmar situado en la esquina inferior izquierda.

El botón definido como volver, es empleado para volver a la pantalla anterior.

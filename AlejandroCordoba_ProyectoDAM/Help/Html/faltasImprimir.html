Imprimir faltas de asistencia

Para poder imprimir las falta de asistencia de un módulo profesional, es necesario seleccionar un módulo profesional.

Si se selecciona algún valor se mostrará abajo la información correspondiente.

El botón definido como imprimir, es empleado para imprimir el registro de una falta de asistencia, si ha superado el límite o no.

El botón definido como volver, es empleado para volver a la pantalla anterior.

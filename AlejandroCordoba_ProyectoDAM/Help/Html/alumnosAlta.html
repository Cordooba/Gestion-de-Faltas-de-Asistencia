Alta de alumnos

Para poder dar de alta o registrar a un nuevo alumno, es necesario introducir todos los campos que se exigen, ya que todos los campos son obligatorios.

Una vez introducido todos los campos, solo deberemos pulsar el boton confirmar para realizar la operación.

El botón definido como volver, es empleado para volver a la pantalla anterior.

El botón definido como limpar campos, se utiliza para facilitar al usuario el borrado del contenido de las cajas de texto.
